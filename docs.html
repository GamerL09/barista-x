Economy

  • !balance – View your wallet and bank balance.
  • !daily – Claim your daily reward.
  • !weekly – Claim your weekly reward.
  • !work – Work a job to earn coins.
  • !beg – Beg strangers for some quick change.
  • !search – Search areas to find hidden coins.
  • !gamble – Gamble coins for a chance to win big.
  • !withdraw – Move coins from bank to wallet.
  • !deposit – Move coins from wallet to bank.
  • !give – Transfer coins to another user.

Fun

  • !meme – Sends a random meme from Reddit.
  • !roast – Roasts a user with a funny insult.
  • !joke – Tells a random joke.
  • !pp – Expose someones pp.
  • !rate – Rates anything you type.
  • !simp – Measures someone's simp level.
  • !8ball – Ask the magic 8-ball a question.

Moderation

  • !ban – Bans a user from the server.
  • !kick – Kicks a user from the server.
  • !mute – Mutes a user so they can't chat.
  • !unmute – Unmutes a muted user.
  • !warn – Warns a user for rule-breaking.

Utility

  • /suggest – Send a suggestion to the server admins.
  • /report – Report a bug or issue with the bot.

Config (Slash Commands)

  • /apply_grinder – Apply to become a grinder in the server.
  • /set_grinder_support – Set support roles for grinder applications.
  • /alert – View the latest server alert.
  • /tickets – Set up the ticket support system.
